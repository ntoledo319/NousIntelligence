TITLE: “Doc-Apocalypse — Rebuild ALL Documentation from Live Code”

OBJECTIVE  
• Assume every existing doc is outdated or wrong.  
• Re-scan the **entire codebase** and auto-generate fresh documentation that mirrors current capabilities, APIs, and data models—without renaming your uniquely-named files.  

TASK LIST  
1. **Pre-flight**  
   – Backup `/docs-backup-$(date)` folder.  

2. **Source Scan & Doc Generation**  
   – JavaScript / TypeScript: run `jsdoc-to-markdown` and/or `typedoc` on all `*.js/ts/tsx` with JSDoc/TSDoc comments.  
   – Python: run Sphinx-auto-api on `*.py`; configure `autodoc` for docstring extraction.  
   – API layer: generate OpenAPI 3.1 JSON & YAML via swagger-jsdoc (Express) or equivalent; produce HTML sandbox.  
   – SQL / ORM models: dump schema diagrams (Mermaid ER or dbml) into `architecture.mmd`.  
   – Combine outputs into Markdown/HTML bundle inside `/generated-docs`.  

3. **Quality Gates**  
   – Run link-checker across new docs; fail build on broken links.  
   – Execute `lighthouse-ci` on the static build; enforce score ≥90 performance/accessibility.  

4. **Doc Injection & Merge**  
   – Locate existing doc files by title and section headers (not filename); splice updated sections in place.  
   – Preserve any bespoke front-matter or formatting quirks.  
   – Append autogenerated CHANGELOG entry describing rebuild date and tools used.  

5. **Continuous Docs Pipeline**  
   – Add GitHub Action: on `push` to `main`, regenerate docs, run link-checker + Lighthouse, and push artifacts to `gh-pages` branch.  
   – Cache JSDoc/TypeDoc outputs for faster CI.  

6. **Final Checkpoint**  
   – Open `/generated-docs/index.html` in preview; verify sidebar, search, and Mermaid diagrams render.  
   – Commit as `docs(rebuild): sync with live codebase`.  

ACCEPTANCE CRITERIA  
✓ All code entities (functions, classes, endpoints, DB tables) documented.  
✓ No broken links or missing references.  
✓ Architecture diagram updated with current components.  
✓ Existing file names untouched; stale sections replaced.  
✓ CI pipeline green on docs regeneration.  

END OF PROMPT
