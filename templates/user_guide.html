Getting Started

NOUS is your AI-powered personal assistant designed to help you manage your daily life through a natural conversation interface. Here's how to get started:

1. Connect Your Accounts

To unlock NOUS's full potential, connect your accounts:

  • Google Account: For calendar, tasks, and email integration
  • Spotify: For music controls and recommendations
2. Explore the Interface

NOUS has a simple and intuitive chat interface. You can:

  • Type commands or questions
  • Use voice commands by clicking the microphone icon
  • View your conversation history
  • Access quick reference guide for commands
3. Try Basic Commands

Start with these simple commands to explore NOUS's capabilities:

  • help - Show available commands
  • what's my day - View today's calendar
  • add task: buy groceries - Add a new task
  • weather - Get current weather

Chat Interface

NOUS's chat interface is designed for natural conversations. You can interact in several ways:

Natural Language

You can use natural language to ask questions or give commands. NOUS will understand your intent and respond appropriately. Examples:

  • "What appointments do I have tomorrow?"
  • "I need to add a dentist appointment next Tuesday at 2pm"
  • "Show me my shopping list for groceries"
  • "How much have I spent on dining this month?"
Structured Commands

For specific functions, you can use structured commands for more precise control:

Command Description Example
add [event] at [time] Create calendar event add meeting with John at tomorrow 3pm
add task: [task] Add task to Google Tasks add task: finish report
add note: [note] Add note to Google Keep add note: Ideas for project
Context-Aware Conversations

NOUS remembers your conversation context, so you can have more natural follow-up interactions:

You: What's the weather like today?

NOUS: Today in [Your Location] it's 72°F and sunny.

You: How about tomorrow?

NOUS: Tomorrow will be 68°F with a 30% chance of rain in the afternoon.

Voice Commands

Speak directly to NOUS using the voice interface:

Using Voice Commands
  1. Click the microphone icon in the command input area
  2. Wait for the "Listening..." indicator
  3. Speak your command clearly
  4. NOUS will process your speech and respond accordingly
Tip: For best results, speak clearly and use the same command structures as you would when typing.
Voice Response

You can enable voice responses in settings so NOUS will speak its responses along with displaying them:

  1. Go to Settings
  2. Enable the "Voice Responses" toggle
  3. Select your preferred voice type and language
