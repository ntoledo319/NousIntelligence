Help Center

Command Interface

General Commands

These commands help you navigate and use basic features of NOUS.

Command Description
help Show this help page
help [category] Show help for a specific category (e.g., "help health")
clear Clear the command log
dashboard Go to the dashboard view

Calendar & Tasks Commands

Manage your calendar events, tasks, and notes.

Command Description
add [event] at [time] Create a calendar event (e.g., "add dentist appointment at tomorrow 2pm")
what's my day Show today's calendar events
show calendar Display upcoming calendar events
add task: [task] Add a task to Google Tasks
show tasks Display your current tasks
complete task [task_id] Mark a task as completed
add note: [note] Add a note to Google Keep
show notes Display your saved notes

Health Commands

Manage doctors, appointments, medications, and health tracking.

Command Description
add doctor [name] Add a new doctor to your records
show doctors Display all your saved doctors
add appointment with [doctor] Schedule a new doctor appointment
show appointments List your upcoming appointments
add medication [name] Add a new medication to track
show medications List all your medications
refill medication [name] Record a medication refill
show medications to refill List medications that need to be refilled
log workout: [details] Log your workout details
log mood: [mood] [details] Track your mood and related notes
pain forecast Get a pain forecast based on weather conditions

Shopping Commands

Manage shopping lists, products to order, and recurring purchases.

Command Description
create list [name] Create a new shopping list
show shopping lists Display all your shopping lists
add item [item] to list [list] Add an item to a shopping list
view list [list_id/name] View items in a specific list
mark list [list] as ordered Mark a shopping list as ordered
set list [list] as recurring Set a shopping list to recur automatically
show due lists Show shopping lists that are due for ordering
add product [name] Add a product to track for reordering
show products Show all tracked products
update product price [product] Update a product's price
set product [product] as recurring Set a product to be reordered automatically
show due orders Show products that are due to be ordered

Finance Commands

Manage budgets, expenses, and recurring payments.

Command Description
create budget [name] Create a new budget category
show budgets Display all your budget categories
budget summary Show a summary of all budgets
add expense [amount] for [category] Add a new expense
show expenses Display all your expenses
add recurring payment [description] Set up a recurring payment
show recurring payments Display all your recurring payments
show upcoming payments Show payments due in the next 30 days
mark payment [id] paid Mark a recurring payment as paid

Travel Commands

Plan trips, manage itineraries, and keep track of travel details.

Command Description
create trip [name] Create a new trip
show trips Display all your planned trips
show upcoming trips Show trips starting in the next 30 days
show active trip Show details for your current active trip
view trip [trip_id/name] View details for a specific trip
add itinerary item to [trip] Add an activity to your trip itinerary
view trip itinerary [trip] View the itinerary for a specific trip
add accommodation to [trip] Add accommodation details to a trip
view trip accommodations [trip] View accommodations for a specific trip
add document to [trip] Add a travel document to a trip
generate packing list for [trip] Create a standard packing list for a trip
view packing list [trip] View the packing list for a specific trip

Media Commands

Control music playback and find content.

Command Description
play [song/artist] Play music on Spotify (requires Spotify connection)
pause Pause currently playing music
next Skip to the next track
previous Go back to the previous track
show aa reflection Display today's AA daily reflection

Natural Language Tips

You can also interact with NOUS using natural language. Here are some examples:

Examples:
  • "Add a dentist appointment on Thursday at 3pm"
  • "Show me my budget for groceries"
  • "I need to refill my medication"
  • "What items do I need to pack for my trip to Paris?"
  • "When is my next appointment with Dr. Smith?"
  • "Add milk and eggs to my grocery list"
  • "I spent $45 on dinner last night"
  • "What's the weather forecast for my trip?"
Tip:

If NOUS doesn't understand your natural language input, try using one of the specific commands listed in the sections above.
